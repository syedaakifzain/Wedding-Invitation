

## Premium 3D Animated Wedding Invitation Website
**For Moied Shabaz Khan & Saniya**

### Architecture
- Section-based fullscreen layout with CSS scroll snapping
- Framer Motion for all animations and scroll-triggered reveals
- Canvas-based particle systems (rose petals, marigolds, confetti, stars)
- Reusable config file for easy name/date/venue editing

### Sections

**1. Loading Screen**
- Elegant golden monogram animation with shimmer effect
- Fades into the envelope scene

**2. Landing – 3D Envelope**
- Centered luxury envelope with gold foil borders on ivory background
- Floating rose petals + warm light particles in background
- Tap to open: 3D flap animation → card slides out → unfolds
- Names "Moied Shabaz Khan & Saniya" appear with glowing gold typography
- Auto-scrolls to next section after reveal

**3. Haldi Ceremony**
- Bright yellow/orange gradient background
- Falling marigold flowers + turmeric splash particles
- 3D rotating event date card, venue slides in
- Floating garland decorations with parallax
- Scroll-triggered text reveals

**4. Nikah Section**
- Dark emerald/royal blue with moonlit sky
- Mosque silhouette, twinkling stars, floating lanterns
- Geometric Islamic pattern borders
- Glassmorphism detail cards (date, time, venue, dua)
- Smooth morph transition from Haldi colors

**5. Reception**
- Dark maroon + champagne gold palette
- Sparkle particles, slow confetti, chandelier glow effects
- Stage-light gradient animations
- Fade-up cards for date/time/venue
- "We await your presence" animated text with shimmer

### UX & Mobile
- Fullscreen snap-scrolling sections (mobile-first)
- Floating music toggle button (with royalty-free audio)
- Scroll progress indicator
- Touch/swipe-friendly interactions
- Premium serif + script typography pairing

### Tech
- React + Tailwind CSS + Framer Motion
- tsparticles for particle effects
- CSS 3D transforms for envelope and card animations
- Howler.js for audio playback
- All event details in a single config file for easy editing
